MKron
=====
-------------------------------------------------------------------------------

>A cron daemon written by **M**atthew **K**unjummen

MKron is written in Go and designed to be simple to write and (roughly)
POSIX compatible.

For details on how it runs and how to use it, check OUTLINE.md

## Dependencies ##

- Go (duh)
- Some sh compatible shell
